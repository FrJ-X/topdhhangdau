# TopDHVN Website

Welcome to the TopDHVN website project! This project is designed to provide information about top universities in Vietnam and includes visual representations of data related to education.

## Project Structure

The project is organized as follows:

```
topdhvn-website
├── public
│   ├── index.html          # Main entry point for the website
│   └── favicon.ico         # Favicon for the website
├── src
│   ├── pages
│   │   ├── introduce.html   # Introduce page with information about the website
│   │   ├── top-dai-hoc.html # Page listing top universities
│   │   └── bieu-do.html     # Page with charts or graphs
│   ├── styles
│   │   └── main.css         # CSS styles for the website
│   └── scripts
│       └── main.js          # JavaScript for interactive functionality
├── package.json             # npm configuration file
├── README.md                # Project documentation
└── .gitignore               # Files and directories to ignore by Git
```

## Features

- **Dark Theme**: The website features a dark background with light text for better readability.
- **Responsive Design**: The layout is designed to be responsive and user-friendly across devices.
- **Interactive Elements**: JavaScript is used to enhance user interaction and dynamic content loading.

## Getting Started

To get started with the TopDHVN website, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd topdhvn-website
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Open the `public/index.html` file in your browser to view the website.
